# 实现书籍卡片点击跳转到借阅页面并传递书籍信息

## 1. 需求分析
- 点击BookCard.vue中的书籍卡片，将当前书籍的所有信息传递给BorrowCard组件
- 路由跳转到BorrowCard页面（路径：/ShowAll/BorrowCard）

## 2. 实现方案
### 2.1 技术选型
- 使用sessionStorage存储书籍信息，避免URL过长问题
- 使用Vue Router进行路由跳转
- 组件间通过sessionStorage传递数据

### 2.2 实现步骤

#### 步骤1：修改BookCard.vue，添加点击事件和路由跳转
- 为书籍卡片添加点击事件`handleBookClick`
- 在点击事件中，将书籍信息存储到sessionStorage
- 使用`this.$router.push`跳转到BorrowCard路由

#### 步骤2：修改BorrowCard.vue，接收并显示书籍信息
- 在`created`或`mounted`钩子中，从sessionStorage获取书籍信息
- 解析JSON字符串为书籍对象
- 在页面中显示书籍信息

#### 步骤3：优化用户体验
- 添加点击反馈样式
- 确保sessionStorage中的数据在组件卸载时清理

## 3. 代码修改计划

### 3.1 BookCard.vue 修改点
```vue
<!-- 为book-card添加点击事件 -->
<div class="book-card" v-for="book in bookList" :key="book.id" @click="handleBookClick(book)">
  <!-- 原有内容 -->
</div>

<!-- 在script中添加handleBookClick方法 -->
<script>
export default {
  // 原有代码
  methods: {
    // 新增方法
    handleBookClick(book) {
      // 存储书籍信息到sessionStorage
      sessionStorage.setItem('currentBook', JSON.stringify(book));
      // 路由跳转到BorrowCard
      this.$router.push('/ShowAll/BorrowCard');
    }
  }
}
</script>

<!-- 添加点击反馈样式 -->
<style scoped>
.book-card {
  /* 原有样式 */
  cursor: pointer;
  transition: all 0.3s ease;
}

.book-card:hover {
  /* 原有样式 */
  transform: translateY(-2px);
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
}
</style>
```

### 3.2 BorrowCard.vue 修改点
```vue
<template>
  <div class="borrow-card-container">
    <h2>借阅书籍信息</h2>
    <div v-if="currentBook" class="book-detail">
      <img :src="currentBook.book_cover" :alt="currentBook.book_name" class="book-cover" />
      <div class="book-info">
        <h3>{{ currentBook.book_name }}</h3>
        <p>作者: {{ currentBook.book_author }}</p>
        <p>编号: {{ currentBook.book_no }}</p>
        <p>库存: {{ currentBook.stock_num }}</p>
        <p>类型: {{ currentBook.book_type }}</p>
        <!-- 借阅表单或其他操作 -->
      </div>
    </div>
    <div v-else class="no-book">
      未选择书籍
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      currentBook: null
    }
  },
  created() {
    // 从sessionStorage获取书籍信息
    const bookStr = sessionStorage.getItem('currentBook');
    if (bookStr) {
      this.currentBook = JSON.parse(bookStr);
    }
  },
  beforeDestroy() {
    // 清理sessionStorage中的数据
    sessionStorage.removeItem('currentBook');
  }
}
</script>

<style scoped>
/* 添加样式 */
.borrow-card-container {
  padding: 20px;
}

.book-detail {
  display: flex;
  margin-top: 20px;
}

.book-cover {
  width: 150px;
  height: 220px;
  margin-right: 30px;
}

.book-info {
  flex: 1;
}

.no-book {
  text-align: center;
  padding: 50px 0;
  color: #999;
}
</style>
```
